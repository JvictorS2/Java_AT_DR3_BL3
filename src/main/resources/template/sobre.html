Sobre a disciplina

Curso focado em desenvolver competências essenciais em Serviços Web e Testes com Java. Preparar local, criar Testes unitários com JUnit, cosumir e desenvolver serviços web (RESTful APIs) usando Spark Java

Competências

  • Preparar o ambiente para desenvolvimento local com Java
  • Desenvolver testes unitários usando JUnit
  • Consumir serviços web (RESTful APIs) com Java
  • Desenvolver serviços web (RESTful API) usando Spark Java

Comentário do Aluno

No começo da disciplina o professor Elberth introduziu o conceito de Teste unitário, utilizando maven e JUnit, teste que podem ser feitos com praticamente quase todo atributo, método e classe para saber se elas estão funcionando corretamente, continuando ele apresentou o FrameWork SpringBoot utilizando o SpringBoot Initializer para construir um projeto do zero, ensinou sobre as camadas de controller, service e repository, e a utilizar uma coleção Map como um pseudo Banco de dados, logo depois iniciou os conceitos de http Get, post e o consumo de API externa com SpringBoot, nos últimos tópicos iniciou o desenvolvimento com o Framework Spark e demonstrou como criar as camadas controller, services e como criar um CRUD utilizando o método GET e uma Collection MAP. Foi um curso repleto de informação e desafios e o considero um dos melhores professores do Instituto Infnet.
